Basics to Web Dev


What is WEB DEVELOPMENT?

Web development is the process of creating, maintaining, and building websites.
Web development primarily consists of:

  • Front-End Development
  • Back-End Development

What Frontend and Backend?

Front-End Development involves the visual and interactive aspects of websites using HTML, CSS, and JavaScript.

Back-End Development focuses on server-side operations, databases, and server logic using languages like Python, Javascript, etc.

What is a website?

A website is a set of web pages you can visit on the internet.

How does a website work?

In the request-response cycle, a client asks a server for data or services, and the server sends back the requested information. This process allows users to access content on the internet. In case of websites, the response is an HTML document.

What makes up a website?

HTML (Hypertext Markup Language) defines the structure of a webpage, like the skeleton.

CSS (Cascading Style Sheets) defines the design or style of the webpage, like skin and clothes.

JS (JavaScript) defines the behavior of the webpage, like making the skeleton move.
